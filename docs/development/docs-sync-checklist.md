# Docs Sync Checklist

## Kapan Dipakai

Gunakan checklist ini saat implementasi, review, dan release.

## Checklist

- Apakah PRD terdampak?
- Apakah rewrite spec terdampak?
- Apakah architecture terdampak?
- Apakah features docs terdampak?
- Apakah workflow terdampak?
- Apakah deployment terdampak?
- Apakah changelog perlu diupdate?

## Aturan

- Perubahan besar belum selesai jika docs belum sinkron.
- Perubahan flow harus dicatat sebelum merge.
- Perubahan keputusan teknis harus masuk ke dokumen yang tepat.
